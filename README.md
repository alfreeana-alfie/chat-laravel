This chat application was inspired by Facebook Messenger. This applicatio has most of the functions that similar to the Facebook Messenger such as:

For Personal Uses:
# 1-1 Chat Messages
# 1-1 Video Call
# 1-1 Voice Call
# Send/Accept Friend Request
# Friend List

For Grouping:
# Group Chat Messsages
# Group Video Call
# Group Voice Call
